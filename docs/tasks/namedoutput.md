# Named Output

## Inputs
* **input**: The data to publish on the named topic

## Outputs
None

## Configuration
* **Name** The name of the topic.
* **Metadata** Additional metadata to associate with the topic.

## Description
A task that takes data from an input and sends it to a named topic.
